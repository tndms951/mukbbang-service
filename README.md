# Project - 먹빵

## 주변 빵집을 소개해주는 빵집

### 개요
우리동네 빵과 빵집을 찾아볼수 있는 Muk-bbang(먹빵) 서비스 프로젝트입니다. 
해당 프로젝트는 반응형 웹이 가능하며, 서버를 이용해 회원가입 및 로그인, 각종 페이지에서 api를 이용해 페이지를 구현시키는 React 프로젝트 입니다.
